### ❌ 발견된 이슈 목록

#### 🚨 **Critical Issues**

##### 이슈 #1: 서명 문서 렌더링 실패 (Canvas 모듈 누락)
- **위치**: 서명 페이지 (/sign/[shortUrl])
- **증상**: "Error: Cannot find module '../build/Release/canvas.node'" 에러로 문서 로딩 실패
- **재현 단계**:
  1. 서명 링크 접근 (예: http://localhost:3000/sign/d0znc6XkV8Vx)
  2. "문서 로딩 중..." 후 에러 발생
  3. "문서를 찾을 수 없음" 메시지 표시
- **예상 원인**: node-canvas 모듈이 설치되지 않음 또는 빌드되지 않음
- **우선순위**: High
- **영향도**: 서명자가 문서에 접근 불가

##### ~~이슈 #1: 서명 영역 선택 기능 오작동~~ ✅ **해결됨**
- **상태**: 재테스트 결과 정상 작동 확인
- **확인된 기능**: 
  - 서명 영역 드래그 앤 드롭 선택: ✅ 정상
  - 영역 좌표 계산: ✅ 정상 ({x: 510, y: 193, width: 243, height: 229})
  - 서명 영역 저장: ✅ 정상

##### 이슈 #2: 기존 계정 로그인 실패
- **위치**: 로그인 페이지
- **증상**: 기존 계정 정보로 로그인 시 "로그인 중 오류가 발생했습니다" 에러
- **테스트 계정**: pb1123love@gmail.com / password
- **재현 단계**:
  1. 로그인 페이지 접속
  2. 기존 계정 정보 입력
  3. 로그인 버튼 클릭
  4. 에러 메시지 표시
- **우선순위**: Medium
- **영향도**: 기존 사용자 접근 제한

#### ⚠️ **Medium Issues**

##### 이슈 #3: 콘솔 에러 로그
- **위치**: 새 문서 탭 전환 시
- **증상**: 브라우저 콘솔에 다음 에러들이 반복 출력됨:
  ```
  🔄 [UI] Loading existing document: [document-id]
  📊 [UI] getDocumentById 결과: {success: false, error: Cannot coerce the result to a single JSON object}
  ❌ [UI] 기존 문서 로드 실패: Cannot coerce the result to a single JSON object
  ```
- **우선순위**: Medium
- **영향도**: 성능 및 디버깅에 영향

##### 이슈 #4: 회원가입 시 비밀번호 요구사항 표시 이슈
- **위치**: 회원가입 페이지 > 비밀번호 필드
- **증상**: "register.passwordRequirement" 텍스트가 그대로 표시됨 (국제화 키 미처리)
- **예상 원인**: 다국어 번역 키 누락
- **우선순위**: Low
- **영향도**: UX 저하

#### 🔍 **Unable to Test (제한된 테스트)**

##### 문서 서명자 flow
- **제한 사유**: 서명 영역 설정 기능이 작동하지 않아 공유 링크 생성 불가

  - 패스워드로 문서 접근 확인

##### 문서 다운로드 기능
- **제한 사유**: 서명 완료된 문서가 없어 다운로드 테스트 불가
- **의존성**: 서명자 flow 완료 후 테스트 가능
