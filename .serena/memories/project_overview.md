# 프로젝트 개요

## 프로젝트 목적
SeukSeuk - 디지털 서명 플랫폼으로, 문서 업로드, 서명, 공유 기능을 제공하는 웹 애플리케이션

## 기술 스택
- **Frontend**: Next.js 14 (React 18)
- **Database**: Supabase 
- **Auth**: Supabase Auth
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **Language**: TypeScript
- **Package Manager**: pnpm (lockfile 존재)

## 주요 기능
- 사용자 인증 (회원가입/로그인)
- 문서 업로드 및 관리
- 디지털 서명 기능
- 다국어 지원 (Language Context)
- 다크모드 지원 (Theme Toggle)
- 반응형 웹 디자인
